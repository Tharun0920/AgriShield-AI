# 🌾 AgriShield AI: Multimodal Farming Assistant

An end-to-end multi-modal Data Science pipeline integrating predictive analytics with Deep Learning and Generative AI to assist in agricultural decision-making.

## 🚀 Project Overview
This project merges three distinct Data Science domains into a single application to solve real-world agricultural challenges. 

## 🛠️ Core Features
* **Crop Disease Scanner (Computer Vision):** A Deep Learning model utilizing MobileNetV2 architecture to detect plant diseases from unstructured leaf imagery.
* **Yield Forecasting (Tabular Regression):** A Random Forest Machine Learning model that analyzes structured environmental time-series data (temperature, rainfall, fertilizer) to predict final crop production.
* **Interactive Dashboard:** A fully deployed Streamlit web interface allowing end-users to interact with the AI models in real-time.

## 💻 Tech Stack
* **Languages:** Python
* **Deep Learning:** TensorFlow / Keras
* **Machine Learning:** Scikit-Learn, Pandas, NumPy
* **Frontend/Deployment:** Streamlit Community Cloud
